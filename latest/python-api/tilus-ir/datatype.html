class tilus.ir.DataType¶
__eq__(other)¶

Compares this data type with another data type.

Parameters:

other (tilus.ir.DataType) – The other data type to compare against.

Returns:

True if the data types are equivalent, False otherwise.

Return type:

bool

property name¶

The name of the data type.

Type:

str

property nbytes¶

The number of bytes in the data type.

Type:

int

property nbits¶

The number of bits in the data type.

Type:

int

property one¶

A constant with value 1 of this data type.

Type:

tilus.ir.Constant

property zero¶

A constant with value 0 of this data type.

Type:

tilus.ir.Constant

constant()¶

Create a constant of this data type.

Parameters:

value (int, float, bool, or str (for string types)) – The value of the constant.

Returns:
