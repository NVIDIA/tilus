Script.fence¶

Fence instructions for memory ordering.

Naming convention follows the PTX fence instruction taxonomy (PTX ISA 9.7.13.4: membar / fence). Each tilus method corresponds to one PTX instruction template with parameters for meaningful variation.

Implemented¶

proxy_async(space)

Bidirectional async proxy fence: fence.proxy.async.{space}

proxy_async_release()

Unidirectional generic-to-async release: fence.proxy.async::generic.release.sync_restrict::shared::cta.cluster
